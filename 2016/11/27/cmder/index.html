cmder简介

Cmder是一个软件包，它被创建出来替代极不满没有漂亮的控制台模拟器的Windows上
它是基于ConEmu与主要配置改革,附带了一个Monokai配色方案

它的官网：http://cmder.net/
它的github地址：https://github.com/cmderdev/cmder

特点

同一个窗口可以含有多个tab选项卡，类似浏览器那样。

自动添加git bash

鼠标点击右下的绿色加号旁，会出现如下菜单，其中有个bash,我们，选择其中的任一项，都会出现如上图的提示，…git-for-windows…
此时我们已经装了git，但是cmder没有找到，所以会报这个错。上图提示有个快捷键 win+alt+t，

按下win+alt+t，出现如下界面，

我们再点击下图中的 add default task,出现提示，然后击是，再点击save setting

此时列表可能多了一些选项，此时没关闭设置窗口，再选择一次bash中的Git bash,就可以打开一个git bash的选项卡

到此你已经打开一个cmd 选项卡，和一个git bash选项卡了
我们这里使用了 add default task 的方式添加重写默认任务，这种方式快速简单
