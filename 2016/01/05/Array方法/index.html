ArrayеҜ№иұЎ

еҸ‘иЎЁдәҺ 2016-01-05 | еҲҶзұ»дәҺ javascript |

еҹәжң¬жҸҸиҝ°

Arrayж—¶javaScriptзҡ„еҶ…зҪ®еҜ№иұЎпјҢеҗҢж—¶д№ҹжҳҜдёҖдёӘжһ„йҖ еҮҪж•°гҖӮз”Ёе®ғеҸҜд»ҘеҲӣе»әж•°з»„гҖӮ

ArrayеҜ№иұЎеұһжҖ§

йқҷжҖҒж–№жі•isArray

Array.isArray() ж–№жі•з”ЁжқҘеҲӨж–ӯжҹҗдёӘеҖјжҳҜеҗҰдёәж•°з»„гҖӮеҰӮжһңжҳҜпјҢеҲҷиҝ”еӣһ trueпјҢеҗҰеҲҷиҝ”еӣһ falseгҖӮ

ECMAScript3зҡ„еҮҪж•°ж–№жі•

ECMAScript3еңЁArray.prototypeдёӯе®ҡд№үдәҶдёҖдәӣеҮҪж•°пјҢиҝҷдәӣеҮҪж•°дҪңдёәд»»дҪ•ж•°з»„зҡ„ж–№жі•йғҪжҳҜеҸҜз”Ёзҡ„гҖӮ

join()

join() ж–№жі•е°Ҷж•°з»„дёӯзҡ„жүҖжңүе…ғзҙ иҝһжҺҘжҲҗдёҖдёӘеӯ—з¬ҰдёІгҖӮ
еҸӮж•°пјҡеҸҜйҖүпјҢз”ЁдәҺжҢҮе®ҡиҝһжҺҘжҜҸдёӘж•°з»„е…ғзҙ зҡ„еҲҶйҡ”з¬ҰпјҢй»ҳи®ӨйҖ—еҸ·гҖӮ

push()е’Ңpop()

pushж–№жі•з”ЁдәҺеңЁж•°з»„зҡ„жң«з«Ҝж·»еҠ дёҖдёӘжҲ–еӨҡдёӘе…ғзҙ пјҢе№¶иҝ”еӣһж·»еҠ еҗҺзҡ„ж•°з»„зҡ„й•ҝеәҰгҖӮ
popж–№жі•з”ЁдәҺеҲ йҷӨж•°з»„зҡ„жңҖеҗҺдёҖдёӘе…ғзҙ пјҢе№¶иҝ”еӣһиҜҘе…ғзҙ 

reverse()

reverseж–№жі•з”ЁдәҺйў еҖ’ж•°з»„дёӯе…ғзҙ зҡ„йЎәеәҸ,иҝ”еӣһж”№еҸҳеҗҺзҡ„еҺҹж•°з»„

concat()

е°Ҷдј е…Ҙзҡ„ж•°з»„жҲ–йқһж•°з»„еҖјдёҺеҺҹж•°з»„еҗҲе№¶,з»„жҲҗдёҖдёӘж–°зҡ„ж•°з»„е№¶иҝ”еӣһ
иҜӯжі•пјҡarray.concat(value1, value2, вҖҰ, valueN)

shift()е’Ңunshift()

shiftж–№жі•з”ЁдәҺеҲ йҷӨж•°з»„зҡ„з¬¬дёҖдёӘе…ғзҙ пјҢе№¶иҝ”еӣһиҜҘе…ғзҙ гҖӮ
unshiftж–№жі•з”ЁдәҺеңЁж•°з»„зҡ„з¬¬дёҖдёӘдҪҚзҪ®ж·»еҠ е…ғзҙ пјҢе№¶иҝ”еӣһж·»еҠ ж–°е…ғзҙ еҗҺзҡ„ж•°з»„й•ҝеәҰ

slice()е’Ңsplice()

sliceж–№жі•з”ЁдәҺжҸҗеҸ–еҺҹж•°з»„зҡ„дёҖйғЁеҲҶпјҢиҝ”еӣһдёҖдёӘж–°ж•°з»„гҖӮ
иҜӯжі•пјҡarray.slice(begin [,end])
еҸӮж•°пјҡbegin,зҙўеј•еӨ„ејҖе§ӢпјҢзңҒз•ҘпјҢеҲҷй»ҳи®Ө0пјӣ end,зҙўеј•еӨ„з»“жқҹпјҢзңҒз•ҘпјҢеҲҷеҸ–еҲ°ж•°з»„жң«е°ҫ

spliceж–№жі•з”ЁдәҺеҲ йҷӨеҺҹж•°з»„зҡ„дёҖйғЁеҲҶжҲҗе‘ҳпјҢе№¶еҸҜд»ҘеңЁиў«еҲ йҷӨзҡ„дҪҚзҪ®ж·»еҠ е…Ҙж–°зҡ„ж•°з»„жҲҗе‘ҳгҖӮе®ғзҡ„иҝ”еӣһеҖјжҳҜиў«еҲ йҷӨзҡ„е…ғзҙ 
иҜӯжі•пјҡarray.splice(start, deleteCount[, item1[, item2[, вҖҰ]]])
еҸӮж•°пјҡstartзҙўеј•еӨ„ејҖе§ӢпјҢи¶…еҮәдәҶж•°з»„зҡ„й•ҝеәҰпјҢеҲҷд»Һж•°з»„жң«е°ҫејҖе§ӢпјҢеҰӮжҳҜиҙҹеҖјпјҢеҲҷиЎЁзӨәд»Һж•°з»„жң«дҪҚејҖе§Ӣзҡ„з¬¬еҮ дҪҚпјӣdeleteCountпјҢж•ҙж•°пјҢз§»йҷӨзҡ„ж•°з»„е…ғзҙ зҡ„дёӘж•°пјҢdeleteCountеӨ§дәҺstartд№ӢеҗҺзҡ„е…ғзҙ зҡ„жҖ»ж•°пјҢеҲҷд»Һ start еҗҺйқўзҡ„е…ғзҙ йғҪе°Ҷиў«еҲ йҷӨпјҲеҗ«з¬¬ start дҪҚпјү

sort()ж–№жі•

sortй»ҳи®ӨжҢүз…§еӯ—е…ёйЎәеәҸжҺ’еәҸпјҢеҺҹж•°з»„е°Ҷиў«ж”№еҸҳгҖӮ
иҜӯжі•пјҡarr.sort([compareFunction])
еҸӮж•°пјҡеҸҜйҖүгҖӮз”ЁжқҘжҢҮе®ҡжҢүжҹҗз§ҚйЎәеәҸиҝӣиЎҢжҺ’еҲ—зҡ„еҮҪж•°гҖӮеҰӮжһңзңҒз•ҘпјҢе…ғзҙ жҢүз…§еӯ—е…ёйЎәеәҸиҝӣиЎҢжҺ’еәҸгҖӮ
иЎҘе……пјҡ compareFunction(a, b)пјҢе°ҸдәҺ0пјҢйӮЈд№Ҳaдјҡиў«жҺ’еҲ—еҲ°bд№ӢеүҚпјӣзӯүдәҺ 0пјҢa е’Ң b зҡ„зӣёеҜ№дҪҚзҪ®дёҚеҸҳпјӣеӨ§дәҺ0пјҢ bдјҡиў«жҺ’еҲ—еҲ°aд№ӢеүҚ

toString()е’ҢtoLocaleString()

toString жҠҠж•°з»„иҪ¬жҚўдёәеӯ—з¬ҰдёІпјҢе№¶иҝ”еӣһз»“жһңгҖӮ
toLocaleStringжҠҠж•°з»„иҪ¬жҚўдёәжң¬ең°ж•°з»„пјҢе№¶иҝ”еӣһз»“жһңгҖӮ

ECMAScript5дёӯзҡ„ж•°з»„ж–№жі•

ECMAScript 5 е®ҡд№үдәҶ9дёӘж–°зҡ„ж•°з»„ж–№жі•пјҢеҲҶеҲ«жҳҜmapгҖҒforEachгҖҒfilterгҖҒeveryгҖҒsomeгҖҒreduceгҖҒreduceRightгҖҒindexOfе’ҢlastIndexOfпјӣ
иҝҷдәӣж–№жі•еҸҜеңЁж•°з»„гҖҒеӯ—з¬ҰдёІгҖҒзұ»ж•°з»„дёҠдҪҝз”Ёпјӣ
еҸӮж•°пјҡ(callback[, thisArg])
callback:currentValueеҪ“еүҚйЎ№пјҲжҢҮйҒҚеҺҶж—¶жӯЈеңЁиў«еӨ„зҗҶйӮЈдёӘж•°з»„йЎ№пјүзҡ„еҖј;indexеҪ“еүҚйЎ№зҡ„зҙўеј•пјҲжҲ–дёӢж Үпјү;arrayж•°з»„жң¬иә«гҖӮ
thisArgпјҡеҸҜйҖүеҸӮж•°пјҢз”ЁжқҘеҪ“дҪңcallback еҮҪж•°еҶ…thisзҡ„еҖјзҡ„еҜ№иұЎ

forEachж–№жі•

forEach() ж–№жі•и®©ж•°з»„зҡ„жҜҸдёҖйЎ№йғҪжү§иЎҢдёҖж¬Ўз»ҷе®ҡзҡ„еҮҪж•°гҖӮ
иҜӯжі•пјҡarray.forEach(callback[, thisArg])

mapж–№жі•

mapж–№жі•еҜ№ж•°з»„зҡ„жүҖжңүжҲҗе‘ҳдҫқж¬Ўи°ғз”ЁдёҖдёӘеҮҪж•°пјҢж №жҚ®еҮҪж•°з»“жһңиҝ”еӣһдёҖдёӘж–°ж•°з»„гҖӮ
иҜӯжі•пјҡarray.map(callback[, thisArg])

filterж–№жі•

иҜӯжі•пјҡarr.filter(callback[, thisArg])

filter() ж–№жі•дҪҝз”ЁжҢҮе®ҡзҡ„еҮҪж•°жөӢиҜ•жүҖжңүе…ғзҙ пјҢе№¶еҲӣе»әдёҖдёӘеҢ…еҗ«жүҖжңүйҖҡиҝҮжөӢиҜ•зҡ„е…ғзҙ зҡ„ж–°ж•°з»„
иҜӯжі•пјҡarr.filter(callback[, thisArg])

everyе’Ңsomeж–№жі•

every() ж–№жі•жөӢиҜ•ж•°з»„зҡ„жүҖжңүе…ғзҙ жҳҜеҗҰйғҪйҖҡиҝҮдәҶжҢҮе®ҡеҮҪж•°зҡ„жөӢиҜ•гҖӮжүҖжңүйҖҡиҝҮиҝ”еӣһtrueпјҢеҗҰеҲҷfalse
иҜӯжі•пјҡarr.every(callback[, thisArg])

some() ж–№жі•жөӢиҜ•ж•°з»„дёӯзҡ„жҹҗдәӣе…ғзҙ жҳҜеҗҰйҖҡиҝҮдәҶжҢҮе®ҡеҮҪж•°зҡ„жөӢиҜ•гҖӮдёҖжңүйҖҡиҝҮе°ұиҝ”еӣһtrue,еҗҰеҲҷfalse
иҜӯжі•пјҡarr.some(callback[, thisArg])

reduceе’ҢreduceRightж–№жі•

reduceж–№жі•е’ҢreduceRightж–№жі•зҡ„дҪңз”ЁпјҢжҳҜдҫқж¬ЎеӨ„зҗҶж•°з»„зҡ„жҜҸдёӘе…ғзҙ пјҢжңҖз»ҲзҙҜи®ЎдёәдёҖдёӘеҖјпјҢreduceеҜ№ж•°з»„е…ғзҙ зҡ„еӨ„зҗҶйЎәеәҸжҳҜд»Һе·ҰеҲ°еҸіпјҢreduceRightеҲҷжҳҜд»ҺеҸіеҲ°е·ҰгҖӮ
иҜӯжі•пјҡarr.reduce(callback,[initialValue])
callbackеҸӮж•°жңүпјҡ
previousValueпјҡдёҠдёҖж¬Ўи°ғз”Ёеӣһи°ғиҝ”еӣһзҡ„еҖјпјҢжҲ–иҖ…жҳҜжҸҗдҫӣзҡ„еҲқе§ӢеҖјпјҲinitialValueпјү
currentValueпјҡж•°з»„дёӯеҪ“еүҚиў«еӨ„зҗҶзҡ„е…ғзҙ 
indexпјҡдёӢж Ү
arrayпјҡи°ғз”Ё reduce зҡ„ж•°з»„
initialValueпјҡдҪңдёәз¬¬дёҖж¬Ўи°ғз”Ё callback зҡ„з¬¬дёҖдёӘеҸӮж•°гҖӮ

indexOfе’ҢlastIndexOfж–№жі•

indexOfиҝ”еӣһз»ҷе®ҡе…ғзҙ еңЁж•°з»„дёӯз¬¬дёҖж¬ЎеҮәзҺ°зҡ„дёӢж ҮпјҢеҰӮжһңжІЎжңүеҮәзҺ°еҲҷиҝ”еӣһ-1
иҜӯжі•пјҡarr.indexOf(searchElement[, fromIndex = 0])

lastIndexOfиҝ”еӣһз»ҷе®ҡе…ғзҙ еңЁж•°з»„дёӯжңҖеҗҺдёҖж¬ЎеҮәзҺ°зҡ„дёӢж ҮпјҢеҰӮжһңжІЎжңүеҮәзҺ°еҲҷиҝ”еӣһ-1пјҢд»Һж•°з»„зҡ„еҗҺйқўеҗ‘еүҚжҹҘжүҫпјҢд»Һ fromIndex еӨ„ејҖе§Ӣ
иҜӯжі•пјҡarr.lastIndexOf(searchElement[, fromIndex = arr.length - 1])
